Twilio SMS setup

Free trial at twilio.com — gives you ~$15 credit (~1,500 texts). No coding required once connected.
  1. Sign up at twilio.com
  2. Go to Console → get your Account SID and Auth Token
  3. Buy a phone number (~$1/month) from the Twilio console
  4. Paste your credentials below and save

Wix embed code

In Wix editor → click Add → Embed → HTML iframe → paste the code below.
<iframe
  src="https://gco-dispatch.vercel.app"
  width="100%"
  height="900px"
  frameborder="0"
  allow="geolocation"
  style="border-radius:12px;border:none"
></iframe>

Replace YOUR-HOSTED-URL with the Vercel or Netlify URL from Step 3. The allow="geolocation" attribute enables GPS tracking inside Wix.
